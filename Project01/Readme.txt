L'objectif du Project01 est de mettre en oeuvre jdbc, pour faire accès à partir d'une application java standalone à une base de données.

J'ai utilisé une base de données locale MySQL sur le software Wampserver64 version 3.2.6,
tout en installant un jar file qui est le driver ("mysql-connector-java-8.0.11.jar").

Vous trouverez 2 print-screens: une qui montre la table voiture sur wamp ("Capture de la table Voiture.png"), l'autre qui indique le résultat obtenu ("Resultat Project01.png").
